# grit-example

> The example generator for learning grit

## Usage

Install [Grit](https://github.com/TimCrooker/projenerator) first.

```bash
yarn global add grit-cli
# or
npm i -g grit-cli
```

### From npm

```bash
grit example my-project
```

### From git

```bash
grit timcrooker/grit-example my-project
```
